# これは何

私の環境(Debian stable)でいい感じに動作するようにしたninix-ayaのforkです。
ついでにninix-aya自体もosdn.jpの存続が怪しいのでその保存の意味もあります。
v0.0.0がninix-aya 5.0.9に対応しているので
必要であればそちらを参照してください。
